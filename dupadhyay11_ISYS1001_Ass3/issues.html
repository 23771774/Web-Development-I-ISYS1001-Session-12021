Website Issues

Fonts / Images

The first issue I encountered during the current website development was the font size / images being too small. When the font size / images are too small it is a design issue and should be corrected for better readability and visibility.

Lack of color

Lack of color is the another issue encountered during website development. If the website is completely based on two colors i.e Black and White then it is not easy to navigate, doesn't look conviencing, etc. More the color more it looks better and please the user who is browsing it.

Multiple columns

Having multiple columns in a webpage makes the website cleaner and simple. Three column design also makes the webpage design looks modern.
